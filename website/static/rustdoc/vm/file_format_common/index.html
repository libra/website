write_u16_as_uleb128

LEB128 is a compressed representation for integer values from the DWARF3 spec, http://dwarfstd.org/Dwarf3Std.php or https://en.wikipedia.org/wiki/LEB128. It's used to compress mostly indexes into the main file format tables. Take a Vec and a value to write to that vector and applies LEB128 logic to compress the u16
