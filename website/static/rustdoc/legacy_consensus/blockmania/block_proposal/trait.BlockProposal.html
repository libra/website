fn max_height(&self) -> Index

We can compute the max height of this proposal, which is the largest round over the transitive closure of references.

It is an explicit purpose of this not to compute the transitive closure at each call, but to rather rely on the max_height indexed in references, as this allows purging memory of committed proposals.
