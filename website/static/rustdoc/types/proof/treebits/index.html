types::proof

[−][src]Module types::proof::treebits

This module provides functions to manipulate a perfect binary tree. Nodes in the tree are represented by the order they would be visited in an in-order traversal. For example --

     3
    /  \
   /    \
  1      5
 / \    / \
0   2  4   6

This module can answer questions like "what is the level of 5" (level(5)=1), "what is the right child of 3" right_child(3)=5
