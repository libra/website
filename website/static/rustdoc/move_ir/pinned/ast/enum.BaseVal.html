pub enum BaseVal {
    CopyableVal(CopyableVal),
    Resource(StructInstance),
    Restricted(StructInstance),
    Array(Vec<BaseVal>),
    Option(Box<Option<BaseVal>>),
}

Lowest of the value hierarchy in the language realm. These values have a literal form and can be stored in structures

Variants

CopyableVal(CopyableVal)Resource(StructInstance)

A linear resource accessible via the methods defined in its struct

Restricted(StructInstance)

A local linear value accessible via the methods defined in its struct Not always copyable as it can hold resources
