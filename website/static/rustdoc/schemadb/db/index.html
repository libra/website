schemadb

[−][src]Module schemadb::db

The purpose of this library is to intercept all calls to the underlying key value store and enforce a set of libra specific options.

This library will also handle custom comparators, any encoding/decoding of higher level data structures based on schema settings to byte arrays, writing them to underlaying rocksdb instance with internal schema-to-column family mapping, and provides schematized api to assist with MVCC (multi-version concurrency control).
