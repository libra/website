fn new(code: &[Bytecode]) -> Result<Self, VMStaticViolation>

Given a vector of bytecodes, constructs the control flow graph for it. Return a VMStaticViolation if we were unable to construct a control flow graph, or if we encounter an invalid jump instruction.

Given a block ID, return the reachable blocks from that block including the block itself.
