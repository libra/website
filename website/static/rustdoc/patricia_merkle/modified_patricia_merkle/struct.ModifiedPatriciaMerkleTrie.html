pub fn validate(&self, root_digest: &HashValue) -> Validity[src]

This method will validate the data integrity of the subtree, whose root node is specified by root_digest.

Returns Validity::Valid if the subtree is valid; Validity::Invalid if the subtree is invalid, so user could retrieve the whole subtree again; Validity::ConditionalValid(invalid_digests) if the subtree is valid on the condition that the nodes with invalid_digests are all valid but they failed to be validated locally. Under such scenario, user may want to just re-fetch smaller subtrees originating from them instead of the whole subtree to save network traffic. Note: if the number of invalid_digests exceeds a limit, thought rarely, due to memory allocation concern, we regard the whole subtree is invalid.
