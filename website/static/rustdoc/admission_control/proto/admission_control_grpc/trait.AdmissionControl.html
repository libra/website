This API is used to update the client to the latest ledger version and optionally also request 1..n other pieces of data. This allows for batch queries. All queries return proofs that a client should check to validate the data. Note that if a client only wishes to update to the latest LedgerInfo and receive the proof of this latest version, they can simply omit the requested_items (or pass an empty list). AC will not directly process this request but pass it to Storage instead.
